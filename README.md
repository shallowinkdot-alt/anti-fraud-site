# 🛡️ 反诈情景实验室

> **知己知彼，让诈骗无处遁形**  
> 一个面向大学生群体的反诈教学与情景模拟项目

---

## 📌 项目简介

反诈情景实验室是一个 **交互式反诈教育平台**，通过高度仿真的诈骗场景模拟，帮助大学生 **亲身体验** 骗局的完整推进过程，从而在真实生活中更敏锐地识别诈骗套路、更冷静地应对突发风险。

**核心宗旨**：  
> 最好的防守，不只是“看过提醒”，而是亲身体验一次骗局是如何推进的。

---

## 🎯 项目背景

全国大学生群体长期处于**兼职赚钱、网络购物、游戏交易、校园通知、账号服务**等高危场景中。骗子的手法不断迭代，而受害原因往往不是“缺乏常识”，而是骗子精准抓住了**时间压力、从众心理、占便宜心理、权威身份和流程依赖**。

反诈情景实验室正是为了填补这一教育空白而生——它不再停留在“看海报、读案例”的被动宣传，而是让用户**在安全的模拟环境中“上当”一次**，从而获得终身免疫。

## 🛠️ 使用说明

1. **访问首页** → 了解当前大学生受骗形势及高频诈骗类型。
2. **点击“开始模拟体验”** → 进入模拟体验中心，选择任意场景。
3. **按照提示完成模拟流程** → 在“假页面”中输入信息（均为模拟，无真实风险）。
4. **体验结束后** → 自动跳转至对应的防骗科普页面，深入学习该场景的识别要点和防护建议。
5. **返回首页或继续体验** → 巩固认知，举一反三。

> ⚠️ 本平台所有模拟均为 **虚构场景**，不涉及真实诈骗行为，请放心体验。
